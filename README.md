# Odin Recipes 

This project is part of **The Odin Project Foundations course**.  
It is a simple website built using basic **HTML** to practice structuring web pages and working with links, lists, and images.

## Project Description

Odin Recipes is a basic recipe website that includes:
- A main index page
- Multiple recipe pages
- Links between pages
- Images and formatted content

The goal of this project is to strengthen my understanding of HTML fundamentals.

## Skills Demonstrated

By completing this project, I have practiced:
- Writing semantic HTML
- Using headings, paragraphs, lists, and links
- Adding images to web pages
- Organizing files and folders
- Using Git and GitHub for version control

## What I Learned

This project helped me understand how real websites are structured and gave me confidence in using HTML without relying on templates or frameworks.

## Future Improvements

In the future, I plan to:
- Add CSS for styling
- Improve layout and design
- Make the website responsive
